Sobre Mí

Hola, soy Ariel Ladner, un estudiante de Ingeniería en Sistemas de Información y apasionado por el desarrollo de software. Con una formación autodidacta y proyectos personales, busco siempre aprender nuevas tecnologías y resolver problemas complejos. Me especializo en Python y desarrollo web, con experiencia en bases de datos y soluciones administrativas.

Habilidades

  • Lenguajes de Programación: Python, C, JavaScript, HTML, CSS
  • Desarrollo de Software: Aplicaciones administrativas, gestión de proyectos, sistemas de votación
  • Bases de Datos: MySQL, SQLite
  • Frameworks: Django, Flask
  • Control de Versiones: Git, GitHub

Proyectos

Proyecto 1: Software Administrativo

Creación de un software administrativo diseñado para facilitar la gestión de diversas tareas según las necesidades de los usuarios. Este sistema permite almacenar información en bases de datos y garantiza un acceso seguro mediante un sistema de inicio de sesión.

Dentro del programa, los "proyectos" se refieren a las diferentes áreas de administración que un usuario puede manejar.
